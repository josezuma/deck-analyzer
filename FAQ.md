# FAQ

## What is deck-analyzer?
CLI that audits pitch decks against YC partner guidelines. Scores content, design, investor-readiness.

## How do I install it?
```bash
git clone https://github.com/josezuma/deck-analyzer.git && cd deck-analyzer
```

## Is this compatible with yc-pitch-deck-max?
Yes. deck-analyzer integrates directly with yc-pitch-deck-max.

## Do I need any external dependencies?
No. All scripts use only Python stdlib.
